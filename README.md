# 🐾 Pet Adoption Portal

A responsive and modern **Pet Adoption Portal Website** built using **HTML, CSS, and Bootstrap**.  
This project helps users browse pets available for adoption and submit adoption requests through a simple and attractive user interface.

---

## 📌 Features

✅ Responsive Navigation Bar  
✅ Attractive Hero Section  
✅ Pet Listing Cards  
✅ Adoption Form  
✅ Fully Responsive Design  
✅ Modern UI with Animations  
✅ Social Media Footer  
✅ Bootstrap Integration  

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4/5
- Font Awesome Icons
